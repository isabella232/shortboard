Importing Schedule Data

Before you can use ShortBoard to view Shortwave broadcast schedules, you must first import the schedule data. Shortboard uses the Tab Delimited Shortwave schedule from PrimeTime Shortwave.

  1. Download the latest Tab Delimited Shortwave schedule from PrimeTime Shortwave.
  2. Open ShortBoard
  3. Select File > Import > File
  4. Navigate to the downloaded schedule file, and select OK.

The schedule will be imported and will replace any existing schedule information.

If you get an error when importing your schedule data, check that:

  • You selected the correct file to import.
  • You download the correct Tab Delimited file from PrimeTime Shortwave.
  • The file was not saved with any HTML markup.
