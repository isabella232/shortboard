ShortBoard

ShortBoard allows you to easily sort, view and filter shortwave broadcast schedules.

Screenshots

Main Interface, Add Broadcast to iCal

Requirements

  • Mac OS X 10.5
  • Tab Delimited Schedule list from Prime Time Shortwave.

ShortBoard has only been tested on Intel Mac OS X 10.5

Release History

.90 Beta (December 17, 2008)

  • Added ability to add a Broadcast to iCal
  • Various bug fixes and optimizations

.85 Beta (December 2, 2008)

  • Initial Public Beta Release
  • Import schedule data from file
  • Filter schedules based on schedule fields
  • Show current broadcasts
  • Display Current time in UTC

Installation

The application is installed like any other Mac application. Simply mount the downloaded DMG file, copy the ShortBoard.app file to your Applications folder, and then launch from that folder.

The appleblog has more information on how to install applications from a DMG file on Mac.
